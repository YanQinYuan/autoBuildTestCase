# AutoBuildTestCase

## Introduce
This a project helps you build test case automatically.

## TDD
## Function

1. if you input like 'varchar(8)', it will automatically build testcase like '12345678','qwe123qweqw'
and declare it True or False.


##　Quick Start.


CHINESE:

1. 根据用户输入生成随机字符串
2. 根据生成的字符串组合成测试用例
3. 保存测试用例到文件里
